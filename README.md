# Split-Screen Login Form - CodingLab

Um formulário de login moderno e responsivo com layout dividido em duas secções (split-screen), desenvolvido exclusivamente com HTML5 e CSS3. O projeto combina uma área lateral com uma imagem ilustrativa abstrata e um painel limpo dedicado à autenticação do utilizador.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do formulário e divisão dos blocos principais de conteúdo.
* CSS3: Estilização completa, uso de variáveis, cantos arredondados e gestão de sombras.
* Flexbox / CSS Grid: Alinhamento e distribuição equilibrada entre a imagem e os campos de login.

## Funcionalidades e Técnicas Aplicadas

* Layout Dividido: Estrutura em duas colunas que separa perfeitamente o design visual (imagem) da área funcional de login.
* Inputs com Ícones Integrados: Campos de entrada para e-mail/telefone e senha com ícones discretos alinhados no canto esquerdo.
* Opções Extras de Autenticação: Inclui caixa de seleção para a função "Remember me" e link direto para "Forgot password?".
* Botão de Ação Destacado: Botão principal de login estilizado com uma cor sólida de alto contraste e cantos suavizados.
* Links de Suporte e Redes Sociais: Chamada inferior para criação de novas contas ("Signup now") acompanhada de ícones minimalistas de suporte.
